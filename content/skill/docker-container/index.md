## 컨테이너 기반 개발 환경

Docker를 사용하여 일관된 개발 환경을 구축하고, 애플리케이션을 효율적으로 패키징하고 배포하는 방법을 학습하고 있습니다.

### 주요 기술
- **Docker Compose**: 멀티 컨테이너 애플리케이션 정의
- **이미지 최적화**: 경량화된 컨테이너 이미지 생성
- **볼륨 관리**: 데이터 영속성 보장
- **네트워킹**: 컨테이너 간 통신 구성

컨테이너 오케스트레이션과 CI/CD 파이프라인 통합을 통해 더욱 효율적인 배포 프로세스를 구현하는 것이 목표입니다.
